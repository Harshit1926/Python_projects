# Python, Machine Learning & Computer Vision Projects

A collection of Python, Machine Learning, and Computer Vision projects built while learning and experimenting with real-world concepts.
These projects explore areas like web development with Flask, image processing with OpenCV, machine learning with Scikit-Learn, and deep learning with PyTorch

# | Project Name                                          | Description                                                                                                                                                                                                                          |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 1. Python Quiz Web App                                | Multiple-choice quiz application built with **Python**, providing instant feedback and score tracking.                                                                                                                               |
| 2. Secret Code Generator                              | Message encryption and decryption tool using custom transformation logic implemented in **Python**.                                                                                                                                  |
| 3. Guess the Number Web App                           | Interactive number guessing game with hints and attempt tracking, built using **Python logic and user input handling**.                                                                                                              |
| 4. Rock Paper Scissors Web App                        | Classic 3-round Rock-Paper-Scissors game against the computer using **Python randomization and conditional logic**.                                                                                                                  |
| 5. Multi Stock Visualizer with 20 Days Moving Average | Financial data visualization project using **Yahoo Finance API, Pandas, and Matplotlib** to plot stock prices and 20-day moving averages.                                                                                            |
| 6. Unit Converter                                     | Python utility tool that converts between **length, weight, and temperature units** with intuitive input handling.                                                                                                                   |
| 7. Flask Camera Capture Web App                       | Web-based camera application built with **Flask and OpenCV**. Users can select internal/external webcams, capture photos, record videos, and save media with timestamp-based filenames through a browser interface.                  |
| 8. Interactive Image Processing Tool                  | Image manipulation tool built using **OpenCV and Python** supporting grayscale conversion, cropping, flipping, rotation, blurring, sharpening, resizing, and saving images.                                                          |
| 9. Smart Face Detector                                | Real-time face, smile, and eye detection system built with **OpenCV Haar Cascades**. Automatically captures images when a smile is detected with cooldown timer and visual overlay feedback.                                         |
| 10. Flu Diagnosis Predictor                           | Machine learning project using **Scikit-Learn Decision Tree Classifier** to predict flu based on symptoms. Includes model evaluation, confusion matrix visualization, and interactive diagnosis interface.                           |
| 11. Face Recognizer (Transfer Learning – MobileNetV2) | Deep learning-based face recognition system built with **PyTorch and pretrained MobileNetV2**. Uses transfer learning, data augmentation, live webcam predictions with OpenCV, model saving, and GPU support.                        |
| 12. Invisibility Cloak using OpenCV                   | Fun **Computer Vision** project inspired by the Harry Potter invisibility cloak effect. Uses **Python, OpenCV, and HSV color masking** to detect cloak colors and replace them with a captured background in real-time webcam video. |
